# 🤖 AI News Bot (n8n + Ollama + Telegram)

An automated AI-powered news summarization bot built using:

- n8n
- Ollama (Llama 3)
- Telegram Bot API
- Google News RSS

## 🚀 Features

- Fetches latest news every 30 minutes
- Summarizes using LLM (Ollama)
- Formats professional Markdown messages
- Sends updates to Telegram group

## 🛠 Tech Stack

- n8n
- Ollama
- Telegram API
- RSS Feed

## 📦 Setup Instructions

1. Install n8n
2. Install Ollama & pull model:

ollama pull llama3

3. Import `ai-news-bot.json` into n8n
4. Configure Telegram credentials
5. Activate workflow
